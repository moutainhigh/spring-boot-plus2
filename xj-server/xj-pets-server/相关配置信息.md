## 启动脚本：执行命令

参考脚本：yabei-server/sh/start.sh
```
# 测试服
/usr/local/work/pets/serverdev/start.sh stop     停止
/usr/local/work/pets/serverdev/start.sh start    启动
/usr/local/work/pets/serverdev/start.sh restart  重启
# 重启并实时运行日志（Alibaba Cloud View自动部署可使用此命令）
/usr/local/work/pets/serverdev/start.sh restart & tail -1000f /usr/local/work/pets/serverdev/log.txt

# 正式服
/usr/local/work/pets/server/start.sh stop     停止
/usr/local/work/pets/server/start.sh start    启动
/usr/local/work/pets/server/start.sh restart  重启
# 重启并实时运行日志（Alibaba Cloud View自动部署可使用此命令）
/usr/local/work/pets/server/start.sh restart & tail -1000f /usr/local/work/pets/server/log.txt

### 实时运行日志
tail -1000f /usr/local/work/yabei/server/log.txt
```



## 查看Linux正在运行的java程序及强制改变

```
ps -ef |grep java 或 ps -aux |grep java 
  
# 查服务端口（进程号）
netstat -anop | grep 11312

# 查看所有端口占用
netstat -tupln 

# 强制停止jar应用（tcp 端口）
sudo fuser -k -n tcp 9048
```
